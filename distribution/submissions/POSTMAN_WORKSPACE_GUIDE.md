# Postman Public Workspace — Quick Setup

## One-Click Import

Your live Postman collection is served at:
```
https://auto-business-agent.replit.app/api/public/postman.json
```

### Steps:
1. Go to https://www.postman.com
2. Create a **Public Workspace** named: `PHANTOM VOICES — AI Voice Clone API`
3. Click **Import** > **Link** > paste your collection URL above
4. The collection auto-populates with:
   - All 10 individual voice requests
   - Search examples (customer support, Japanese, gaming)
   - 5 AI recommendation examples (Vapi, Retell, Bland, LangChain, audiobook)
   - OpenAPI spec and manifest links
5. Click **Publish** to make it discoverable on Postman Explore

## Why This Matters
- Postman Explore has millions of developers browsing public APIs
- Developers can **fork** your collection and start testing immediately
- The "Run in Postman" button can be embedded on your portfolio page
- Every request includes the affiliate signup link in the collection description

## Collection Contents
- **Voice Catalog** (11 requests): List all + individual V1-V10
- **Search & Discovery** (3 requests): Customer support, Japanese language, gaming
- **AI Recommendation** (5 requests): Platform-specific examples for Vapi, Retell, Bland, LangChain, and general audiobook
- **Specs & Discovery** (2 requests): OpenAPI spec + Agent manifest
